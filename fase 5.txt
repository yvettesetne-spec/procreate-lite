## 2. Estabilización de Trazos (Streamline)

Suavizado en tiempo real para trazos tembloros. Esencial para stylus genérico.

#### [MODIFY] [index.html]
- Añadir slider de estabilización en la barra lateral izquierda (debajo del slider de tamaño).
- Valor de 0 (sin estabilización) a 100 (máxima estabilización).

#### [MODIFY] [style.css]
- Slider vertical con icono de onda/corazón.

#### [MODIFY] [main.js]
- **Algoritmo: Moving Average con Catmull-Rom Spline**
  - `stabilizationLevel: 0-100` (default 0)
  - Cuando stabilizationLevel > 0:
    - Mantener buffer de últimos N puntos (N = 3 + Math.floor(stabilizationLevel / 10))
    - En cada pointermove, añadir punto al buffer
    - Calcular punto suavizado como promedio ponderado de los últimos N puntos
    - Dibujar hasta el punto suavizado (no el punto crudo)
    - El último punto del buffer se usa como "punto de anclaje" para el siguiente segmento
  - Para curvas: usar Catmull-Rom spline entre puntos suavizados para curvas naturales
  - **Latencia intencional**: el trazo se dibuja ~2-3 puntos detrás del cursor real, creando la sensación de "tinta fluyendo"
- **Optimización:**
  - Solo activo durante drawing, no en idle
  - Buffer limitado a 10 puntos máximo
  - Cálculo simple: promedio ponderado con peso exponencial decreciente

## 5. Liquify / Warp

Herramienta para distorsionar partes de la imagen.

#### [MODIFY] [index.html]
- Añadir botón de liquify en la barra superior (`ri-magic-line`).
- Panel de configuración de liquify con:
  - Tamaño del pincel (reutilizar slider de tamaño)
  - Selector de modo: Empujar (Push), Arrastrar (Pull), Girar (Twirl), Suavizar (Smooth)
  - Slider de fuerza (1-100)

#### [MODIFY] [style.css]
- Panel de configuración de liquify.

#### [MODIFY] [main.js]
- **Liquify Push:**
  - En pointermove:采样 un área circular alrededor del cursor
  - Desplazar los píxeles en la dirección del movimiento del cursor
  - Algoritmo: para cada pixel dentro del radio del pincel:
    - Calcular distancia al centro del cursor
    - Factor de influencia = 1 - (distancia / radio)² (suave)
    - Desplazar pixel en la dirección del vector de movimiento × factor × fuerza
  - Usar `getImageData` / `putImageData` para manipular pixels
- **Liquify Pull (逆方向):**
  - Misma lógica pero los píxeles se mueven HACIA el cursor en vez de en la dirección del movimiento
- **Liquify Twirl:**
  - En lugar de desplazar en línea recta, rotar los píxeles alrededor del centro del cursor
  - Ángulo de rotación = fuerza × (1 - distancia/radio)
- **Liquify Smooth:**
  - Promediar cada pixel con sus vecinos (blur local)
  - Reduce distorsiones existentes
- **Implementación técnica:**
  - Trabajar sobre una copia del canvas activo (snapshot temporal)
  - En cada pointermove: restaurar snapshot → aplicar liquify completo desde los puntos acumulados
  - Al pointerup: commit el resultado final a la capa activa
  - **Rendimiento**: Para iPad 5th gen, limitar radio máximo a 100px y usar paso de muestreo (cada 2do pixel)
- **Estado:**
  - `liquifyMode: 'push' | 'pull' | 'twirl' | 'smooth'`
  - `liquifyStrength: 1-100`
  - `isLiquifying: boolean`
  - `liquifyPoints: [{ x, y, dx, dy }]` (historial de movimiento)
  - `liquifySnapshot: HTMLCanvasElement` (copia temporal del layer)

## Verification Plan

### Manual Testing (iPad 5th gen)
1. **Selección**: Dibujar algo → seleccionar zona → mover contenido → verificar que se mueve correctamente.
2. **Selección + Escalar**: Seleccionar → arrastrar esquina → verificar escalamiento proporcional.
3. **Selección + Rotar**: Seleccionar → arrastrar handle de rotación → verificar rotación.
4. **Flip**: Seleccionar → flip horizontal → verificar espejado.
5. **Estabilización**: Subir slider a 50 → dibujar curva → verificar suavizado sin temblor.
6. **Estabilización 0**: Bajar a 0 → verificar respuesta inmediata sin delay.
7. **Blend modes**: Crear 2 capas → cambiar blend mode → verificar efecto visual en tiempo real.
8. **Composite view**: Cambiar opacidad de capa con blend mode multiply → verificar que se ve correctamente.
9. **Texto**: Activar texto → escribir → cambiar fuente/tamaño → confirmar → verificar texto rasterizado.
10. **Liquify Push**: Dibujar forma → activar liquify → arrastrar sobre la imagen → verificar deformación.
11. **Liquify Twirl**: Cambiar a twirl → mantener pulsado → verificar rotación local.
12. **Liquify Smooth**: Distorsionar → cambiar a smooth → pasar sobre zona → verificar que se suaviza.
13. **Undo post-liquify**: Hacer liquify → undo → verificar restauración completa.
14. **Auto-save**: Hacer selección + transform → refrescar → verificar que se conserva.
